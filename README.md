# 🏟️ PromptArena

### _Where AI Models Compete_

![Status](https://img.shields.io/badge/status-planning-orange)

---

## 📖 About

**PromptArena** is a full-stack web application that allows users to compare responses from multiple AI models simultaneously. Send one prompt and get responses from **Claude**, **GPT**, and **Gemini** side-by-side.

> ⚠️ **Note:** This project is currently in the planning phase. Development will begin soon.

---

## ✨ Planned Features

- 🏟️ Side-by-side comparison of 3 AI models
- ⚡ Live streaming responses
- 🗳️ Voting system for best answers
- 📊 Analytics dashboard
- 🎯 Smart category detection
- 📚 Prompt templates library
- 🧠 Prompt optimization suggestions

---

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, React, Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** MongoDB Atlas
- **AI APIs:** OpenAI (GPT), Anthropic (Claude), Google AI (Gemini)
- **Deployment:** Vercel
